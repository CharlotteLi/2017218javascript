- 同步和异步：
同步：每次只能完成一个任务，必须等这个任务完成之后，才能开始下个任务；
异步：当前的任务没完成，不用等待，继续开始下个任务，也就是，可以多个任务并行；
    + 回调异步
    + 事件
    + 定时器
    + ajax
- http响应状态码
    + 2xx 成功
    + 3xx 重定向
    + 4xx 请求错误
        + 400 请求的参数错误；
        + 404 文件没找到
    + 5XX 服务器错误；
